🐍 Python Script Executor

Führe Python-Code aus, installiere Pakete und verwalte deine `.py`-Dateien in einer benutzerfreundlichen GUI.

✨ Features

  • 🧠 **Code Ausführen**: Direkt Python-Code ausführen und Konsolenausgaben anzeigen.
  • 📦 **Paketinstallation**: Python-Pakete direkt aus der GUI installieren.
  • 📋 **Installierte Pakete anzeigen**: Zeigt alle installierten Pakete an.
  • 💾 **Speichern und Laden von Code**: Speichere oder lade Python-Code als `.py`-Datei.
  • 📚 **Einsteigerfreundliches Tutorial**: Ein Tutorial für Anfänger.
  • 🎨 **Modernes Darkmode UI**: Stilvolles Design basierend auf `tkinter`.
